Tutorials

These lessons will give a step by step introduction to the commands, the classes and the general functionality available in ug4's scripting facility.

The files for these tutorials can be found in the Tutorial app.

  • Tutorial 01 - lua Basics

    This tutorial introduces the basic concepts of the Lua scripting language. It is intended for users who are new to Lua and want to learn the basics. In contrary to the following scripting tutorials, it will not yet use any command specific to ug4.

  • Tutorial 02 - Loading a Domain

    This tutorial will give you a first idea on how to use a lua script to steer ug. We will examine how to use command line arguments to specify filename and a dimension and on how to load and save a domain. Furthermore we will distribute the domain to all involved processes.
